Useful commands

Fedora

  • Update grub:

    $ sudo grub2-mkconfig -o /boot/grub2/grub.cfg

Git

  • Change the git author identity:

    $ git config user.name "author_name"

    $ git config user.email "author_email"

    or to change for all the users:

    $ git config --global user.name "author_name"

    $ git config --global user.email "author_email"

  • Remove a file from the entire commit history:

    $ git filter-branch --tree-filter 'rm -f [file]' -- --all

  • Change the author in the entire commit history:

    $ git filter-branch -f --env-filter "GIT_AUTHOR_NAME='new_name'; GIT_AUTHOR_EMAIL='new_email'; \
      GIT_COMMITTER_NAME='new_name'; GIT_COMMITTER_EMAIL='new_email';" HEAD;

    $ git log   # to check the changes

    $ git push --force origin master

Anaconda

  • Add kernel to Jupyter Notebook:

    $ conda install -c anaconda ipykernel

    $ python -m ipykernel install --user --name=[environment]

  • Remove kernel from Jupyter Notebook:

    $ jupyter kernelspec uninstall [kernel]

  • List all kernels:

    $ jupyter kernelspec list

Tar

  • Archive and compress a directory to a file:

    $ tar -czf file_name.tar.gz dir_name

  • Extract a file:

    $ tar -xzf file_name.tar.gz
